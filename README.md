🧩 Caça-Palavras Estático em C



Projeto desenvolvido para a disciplina de Programação I do curso de Engenharia Elétrica (UFES).



🚀 Funcionalidades



\* Suporte para 2 jogadores com sistema de pontuação.

\* Geração de tabuleiro aleatório via semente (srand).

\* Persistência de dados: Salvar e carregar tabuleiro em arquivos .txt.

\* Busca de palavras em três direções: Horizontal, Vertical e Diagonal.



🛠️ Tecnologias

\* Linguagem C.

&#x09;

&#x09;Bibliotecas: stdio.h, stdlib.h, string.h, time.h.
